This is a static HTML site (no build step) designed for fast hosting and easy edits.

## How to run locally
Open `index.html` in a browser, or run a quick local server:

- Python:
  python3 -m http.server 8000

Then visit http://localhost:8000

## How to deploy (easy)
- Netlify Drop: drag-and-drop the folder.
- Vercel: deploy as a static site.
- GitHub Pages: push the folder and enable Pages.

## Editing your portfolio
Open `data.js` and add/remove items under:
- `singing` (Live / TV / Music Video)
- `content` (Destinations / Resorts / Airlines / Campaigns / Ambassador)

## Images
Images are in `assets/curated/`. Replace them with higher-res originals anytime (keep filenames the same).
